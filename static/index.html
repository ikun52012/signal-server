Multi-Model Voting Experimental

When enabled, multiple AI models analyze each signal concurrently and results are aggregated by voting strategy.

Enter one model per line. Use 'local' for rule-based fallback without API call.

Weighted: avg confidence with recommendation vote. Consensus: majority must agree. Best: highest confidence wins.

Format: model_id: weight. Should sum to ~1.0

Cost Warning: Voting calls multiple APIs per signal, increasing costs. Use local fallback to reduce API calls.
